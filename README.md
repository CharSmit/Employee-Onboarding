# Employee Onboarding Automation System

An automated onboarding solution designed to streamline user
provisioning by integrating directly with **FreshService** and
**Microsoft Entra ID**.

## Overview

This system retrieves onboarding ticket data from FreshService and
automatically creates new user accounts in Microsoft Entra ID using the
parameters provided. It reduces manual effort, improves accuracy, and
ensures a consistent onboarding process.

## Key Features

-   **FreshService Integration**\
    Retrieves onboarding ticket information and uses it to populate new
    user profiles.

-   **Automated Entra Account Creation**\
    Creates Microsoft Entra ID accounts that match the specifications
    defined in the ticket.

-   **Customisable Group Assignment**\
    Administrators can configure how ticket parameters map to security
    groups or roles.

-   **Secure Access Control**\
    Only authenticated tenant users with appropriate permissions can
    access or use the system.

-   **Automatic Population of User Data**\
    User details such as name, department, job title, and other metadata
    are filled in automatically.

## Benefits

-   Reduces manual workload for IT teams\
-   Ensures consistent and compliant account creation\
-   Minimises risk of human error\
-   Speeds up onboarding for new employees
